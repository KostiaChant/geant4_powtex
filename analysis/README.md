# POWTEX Analysis Scripts

This directory contains ROOT analysis scripts for processing POWTEX detector simulation data.

## Overview

The analysis workflow converts raw Geant4 simulation output into calibrated physics data for neutron scattering analysis. The pipeline processes detector events, applies coordinate transformations, removes double-counting, and calculates scattering observables (angles, wavelength, d-spacing).

## Analysis Pipeline

```
Geant4 Simulation (Powtex)
         ↓
    myfile_info.txt  →  [1] CreatePowtexTree.C  →  powtex.root
         ↓
LookupTablePowtex.txt →  [2] ReadLookupTable.C  →  powtex_lookup.root
         ↓
    powtex.root      →  [3] analysis_powtex.C   →  powtex_new + powtex_new_cal.root
         ↓
   [Optional]        →  [4] Extract_branches.C  →  data_powtex_biosample.csv
```

## Scripts

### 1. CreatePowtexTree.C

**Purpose**: Converts raw text data from Geant4 simulation to ROOT format.

**Input**: `myfile_info.txt` (generated by Geant4 simulation)

**Output**: `powtex.root` containing `powtex_ev` TTree

**Usage**:
```bash
root -l -q 'analysis/CreatePowtexTree.C'
```

**Data fields**:
- `tof`: Time of flight (ns)
- `step_no`: Geant4 step number
- `coat`: Event origin (10=cathode, 20=housing)
- `ss_no`: Detector system ID (7=Mantle)
- `sec_no`: Sector number
- `module`: Module number
- `seg`: Segment number
- `voxel`: Gas voxel number
- `scopy`: Counter ID (1-4)
- `energy`: Energy deposited in gas (keV)
- `weight`: Neutron weight (probability)
- `posx, posy, posz`: Detection position (mm)

---

### 2. ReadLookupTable.C

**Purpose**: Creates indexed ROOT database from voxel coordinate lookup table.

**Input**: `LookupTablePowtex.txt` (generated by Geant4)

**Output**: `powtex_lookup.root` containing `lookup_tree` TTree

**Usage**:
```bash
root -l -q 'analysis/ReadLookupTable.C'
```

**Lookup table structure**:
- Maps detector IDs (sumo, module, segment, wire, strip, counter) to voxel center coordinates
- Uses composite indices for fast lookup: `indexi` and `indexj`
- Handles coordinate transformations for Mantle detector geometry

---

### 3. analysis_powtex.C

**Purpose**: Main physics analysis pipeline - processes raw events and calculates scattering observables.

**Input**:
- `powtex.root` (from CreatePowtexTree.C)
- `powtex_lookup.root` (from ReadLookupTable.C)

**Output**:
- `powtex.root` (updated with `powtex_new` tree)
- `powtex_new_cal.root` (calibration tree)

**Usage**:
```bash
root -l -q 'analysis/analysis_powtex.C'
```

**Processing steps**:
1. Reads raw detector events
2. Maps voxel IDs to wire/strip coordinates using helper functions:
   - `calculateW_for_Mantel()` - calculates wire number
   - `calculateS_for_Mantel()` - calculates strip number
3. Removes double-counting (events within 10 ns in same detector element)
4. Applies energy resolution smearing (10% FWHM Gaussian)
5. Calculates physics quantities:
   - `twotheta`: Scattering angle θ (degrees)
   - `phi`: Azimuthal angle φ (degrees)
   - `lambda`: Neutron wavelength (Å)
   - `d_spacing`: Crystal d-spacing (Å)
6. Creates calibration tree with voxel center coordinates

**Output trees**:

**powtex_new** branches:
- Time: `ntof`
- Detector IDs: `nsumo`, `nsegment`, `nmodule`, `ncounter`, `nwire`, `nstrip`
- Indices: `nindexS`, `nindexC`
- Energy: `nenergy` (with resolution smearing)
- Position: `nposx`, `nposy`, `nposz`
- Physics: `lambda`, `twotheta`, `phi`, `d_spacing`, `rad`
- Weight: `nweight`

**powtex_new_cal** branches:
- Indices: `nikS`, `nikC`
- Voxel centers: `nvoxel_x`, `nvoxel_y`, `nvoxel_z`
- Physics: `lambda_c`, `twotheta_c`, `phi_c`, `d_spacing_c`, `rad_c`
- Counter: `nc`
- Weight: `cweight`

---

### 4. Extract_branches.C

**Purpose**: Exports processed ROOT data to CSV format for external analysis.

**Input**:
- `powtex.root` (`powtex_new` tree)
- `powtex_new_cal.root` (friend tree)

**Output**: `data_powtex_biosample.csv`

**Usage**:
```bash
root -l -q 'analysis/Extract_branches.C'
```

**CSV columns**:
- tof [ns]
- lambda [Å]
- module, segment, counter, wire, strip
- x_pos, y_pos, z_pos [mm]
- voxel_x, voxel_y, voxel_z [mm]

## Quick Start

### Option 1: Automated Workflow

Run the complete analysis pipeline:
```bash
./scripts/run_full_analysis.sh
```

This runs all steps automatically and moves output files to `data/output/`.

### Option 2: Manual Step-by-Step

```bash
# 1. Run simulation
./build/Powtex macros/run.mac

# 2. Convert to ROOT
root -l -q 'analysis/CreatePowtexTree.C'

# 3. Build lookup table
root -l -q 'analysis/ReadLookupTable.C'

# 4. Run physics analysis
root -l -q 'analysis/analysis_powtex.C'

# 5. Optional: export to CSV
root -l -q 'analysis/Extract_branches.C'
```

For interactive analysis and visualization:

```bash
root -l data/output/powtex.root
```

```cpp
// Load the tree
TTree* tree = (TTree*)_file0->Get("powtex_new");
tree->AddFriend("powtex_new_cal", "data/output/powtex_new_cal.root");

// Example plots:

tree->Draw("d_spacing_c>>h(3500,0,7)", "cweight", "hist");
```

## Physics Calculations

### Scattering Angle (2θ)
```cpp
rad = sqrt(x² + y² + z²)
twotheta = 180 - acos(-z/rad) * 180/π
```

### Azimuthal Angle (φ)
```cpp
phi = 2 * atan(x/rad) * 180/π
```

### Neutron Wavelength (λ)
```cpp
lambda = (h/m_n) * tof / (L_0 + rad)
```
Where:
- h = Planck constant
- m_n = neutron mass
- L_0 = 41410 mm (sample-to-source distance)
- rad = sample-to-detector distance

### d-spacing
```cpp
d_spacing = λ / (2 * sin(θ))
```
(Bragg's law)

## Notes

- All scripts must be run from the project root directory
- Include paths are relative: `../include/`
- Energy resolution: 10% FWHM Gaussian smearing applied
- Double-counting removal: events within ±10 ns in same voxel are removed
- The lookup table provides voxel center coordinates for improved position resolution

## Dependencies

- ROOT (to be tested with ROOT 6+!)
- Geant4 output files
- Header files in `../include/`:
  - `S_for_Mantel.h` - Strip calculation functions
  - `W_for_Mantel.h` - Wire calculation functions
  - `const_for_POWTEX.h` - Physical constants

## Troubleshooting

**Error: "File not found"**
- Ensure you're running from the project root directory
- Check that Geant4 simulation completed successfully
- Verify `myfile_info.txt` and `LookupTablePowtex.txt` exist

**Error: "Cannot find tree powtex_ev"**
- Run `CreatePowtexTree.C` first
- Check that `powtex.root` was created successfully

**Error: "Include file not found"**
- Verify header files exist in `../include/` relative to analysis directory
- Check file permissions

**No events in output**
- Check simulation ran successfully
- Verify detector geometry is correctly defined
- Check neutron source configuration in macros
